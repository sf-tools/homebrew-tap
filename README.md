# Highjeans Tap

## How do I install these formulae?

`brew install sf-tools/tap/<formula>`

Or `brew tap sf-tools/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "sf-tools/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
